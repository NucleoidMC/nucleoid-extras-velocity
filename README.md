# nucleoid-extras-velocity

## Example Config
Replace `plugins/nucleoid-extras-velocity/config.json` with this, and fill in the settings:
```json
{
    "integrations": {
        "host": "localhost",
        "port": 12345,
        "channel_name": "proxy",
        "enabled": [
            "commands",
            "lifecycle"
        ]
    }
}
```
